# 🖥️ Tiny Cowork: All-in-One GUI App Vision

Tiny MoA(Tiny Cowork)를 터미널 환경(CUI/TUI)을 넘어, 비개발자들도 클릭만으로 강력한 AI 협업군단을 거느릴 수 있는 **독립된 애플리케이션**으로 진화시키기 위한 UX/UI 비전입니다.

---

## 🎨 1. 사용자 경험 비전 (UX Vision)

"터미널이 아닌, **지능형 스튜디오**처럼 느껴지는 경험"

### 📍 핵심 컨셉
- **No-Terminal Flow**: 설치부터 실행까지 검은 화면을 볼 필요가 없는 완전한 GUI 환경.
- **VS Code-like Productivity**: 좌측에는 파일/에이전트 탐색기, 중앙에는 작업 공간, 우측에는 실시간 상태 창이 있는 고효율 레이아웃.
- **Visual Orchestration**: 에이전트들이 서로 대화하고 협업하는 과정을 시각적인 '커넥션 맵'이나 '대화 버블'로 직관적으로 표시.

---

## 🛠️ 2. 주요 GUI 기능 (App Features)

### 🧩 에이전트 컨트롤 센터
- **드래그 앤 드롭 태스크**: 사용자가 원하는 에이전트를 작업 보드에 끌어다 놓아 협업 구조를 시각적으로 설계.
- **에이전트 대시보드**: 각 에이전트의 현재 부하, 강점, 상태를 한눈에 파악.

### 📝 지능형 통합 편집기
- **실시간 마크다운 프리뷰**: 에이전트가 작성하는 보고서나 코드를 실시간으로 렌더링하여 확인.
- **인라인 수정 가이드**: 결과물의 특정 부분을 클릭하면 해당 작업을 수행한 에이전트에게 즉시 수정을 요청하는 팝업.

### ⚙️ 원클릭 환경 구성
- **자동 모델 로더**: 사용자의 사양에 맞춰 최적의 모델(LFM 등)을 추천하고 자동으로 다운로드 및 설정.
- **프로젝트 템플릿**: '코드 리뷰', '보도자료 작성', '기술 리서치' 등 목적에 맞는 에이전트 조합을 템플릿으로 제공.

---

## 🚀 3. 아키텍처 및 기술 스택 (Proposed Tech Stack)

- **Frontend**: **Next.js + TailwindCSS + Shadcn/ui** (미려하고 반응성이 뛰어난 인터페이스)
- **Desktop Wrapper**: **Electron** 또는 **Tauri** (로컬 시스템 파일 및 도구 접근을 위한 데스크톱 패키징)
- **Communication**: Local Python 서버와 Fast API/WebSocket을 통한 실시간 데이터 스트리밍.

---

## 📈 4. 단계별 앱 로드맵 (App Roadmap)

### 1단계: 웹 기반 프로토타입 (Web Dashboard)
- 현재의 TUI 기능을 브라우저로 옮겨 실시간 로그와 결과물을 시각화하는 단계.

### 2단계: 데스크톱 애플리케이션 (Desktop App)
- 파일 시스템과 연동되어 VS Code처럼 자신의 폴더를 열고 에이전트에게 명령을 내리는 전용 앱 배포.

### 3단계: 지능형 협업 스튜디오 (AI Studio)
- 에이전트 스토어와 연동되어 외부 에이전트를 쇼핑하고 자신의 앱에 즉시 추가하는 완전한 생태계 통합.
